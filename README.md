# FoodMatcher-Python

这是一个Python项目：Word2Vec食物推荐模型

## 具体功能

系统预设几种食物，每种食物都有一个至多个标签，用户输入文字，系统匹配食物

## 环境

使用Anaconda

`conda install python`

`conda install gensim`

`conda install -c conda-forge jieba`

## 用到的包：

gensim==4.4.0

numpy==2.4.4

jieba==0.42.1
